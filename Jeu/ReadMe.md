# LAN Blaster

## Compilation

Linux :

make

## Lancer le serveur

./server

## Lancer le client

./client

## Contrôles

ZQSD : déplacement
ESPACE : tirer

## Fonctionnement

Le serveur gère les positions, collisions et tirs.
Les clients affichent le jeu.

## Réseau

Le jeu fonctionne en réseau local UDP.
